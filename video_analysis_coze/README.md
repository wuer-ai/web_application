# 视频解析网站

这是一个基于Django开发的视频解析网站，使用Coze API实现视频解析功能。网站支持用户注册、登录、积分系统等功能，让用户可以方便地解析各大视频平台的视频。

## 功能特点

### 1. 用户系统
- **注册功能**
  - 支持用户名、邮箱、密码注册
  - 注册成功后自动登录
  - 新用户注册赠送10积分
  - 密码强度验证

- **登录功能**
  - 支持用户名密码登录
  - 登录成功后自动跳转到首页
  - 每日首次登录奖励10积分
  - 记住登录状态

- **退出功能**
  - 安全退出系统
  - 退出后跳转到首页

### 2. 积分系统
- 新用户注册赠送10积分
- 每日首次登录奖励10积分
- 每次解析视频消耗1积分
- 积分不足时无法解析视频
- 实时显示用户积分

### 3. 视频解析
- 支持解析B站、优酷、腾讯等主流视频平台
- 解析结果包含：
  - 视频标题
  - 视频封面
  - 播放链接
  - 频道/UP主信息
  - 视频类型
- 支持分集视频解析
- 解析历史记录保存

### 4. 个人中心
- 显示用户基本信息
- 显示当前积分
- 查看解析历史记录
- 积分使用说明

## 安装说明

1. 克隆项目到本地：
```bash
git clone [项目地址]
cd video_analysis_coze
```

2. 安装依赖：
```bash
pip install django requests
```

3. 初始化数据库：
```bash
python manage.py makemigrations
python manage.py migrate
```

4. 创建超级用户：
```bash
python manage.py createsuperuser
```

5. 收集静态文件：
```bash
python manage.py collectstatic
```

6. 启动开发服务器：
```bash
python manage.py runserver
```

7. 访问网站：
打开浏览器访问 http://127.0.0.1:8000/

## 使用说明

### 1. 游客访问
- 可以访问首页
- 查看使用说明
- 无法使用解析功能

### 2. 用户注册
1. 点击导航栏的"注册"按钮
2. 填写用户名、邮箱和密码
3. 提交注册表单
4. 注册成功后自动登录并跳转到首页

### 3. 用户登录
1. 点击导航栏的"登录"按钮
2. 输入用户名和密码
3. 点击登录按钮
4. 登录成功后自动跳转到首页

### 4. 视频解析
1. 登录后访问首页
2. 在输入框中粘贴视频链接
3. 点击"解析视频"按钮
4. 等待解析完成，查看结果

### 5. 查看个人资料
1. 登录后点击导航栏的用户名
2. 查看个人资料和积分信息
3. 查看解析历史记录
